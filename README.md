# deepga
A Deep-Genetic Algorithm (Deep-GA) Approach for High-Dimensional Nonlinear Parabolic Partial Differential Equations

This is a repository for codes and results for the above article. The DeepGA files is the code for our proposed method, while the DeepBSDE files is the code obtained from Jiequn Hana, Arnulf Jentzen, and Weinan E (Solving high-dimensional partial differential equations using deep learning). The term BS and HJB represent the Black-Scholes and Hamilton-Jacobi-Bellman cases used in the research.

Full detailed results of different parameters from both methods for the Black-Scholes and Hamilton-Jacobi-Bellman cases are provided in folder Result.
